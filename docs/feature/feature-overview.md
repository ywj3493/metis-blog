# Feature Overview

This document summarises the user-facing capabilities of Meti's blog without diving into implementation details.

## Blog Home
- Greets visitors with a hero profile section introducing Meti.
- Highlights selected posts in the featured grid for quick discovery.

## Post Browsing
- Dedicated posts page lists every published article sourced from Notion.
- Visitors can filter posts by tag to narrow content quickly.
- Each post detail page renders the Notion content faithfully and links to the previous/next article for continuous reading.

## AI-Powered Summary
- Readers can request an instant AI summary of a post from the detail page.
- Once generated, the summary is stored in Notion so it appears for future visitors without reprocessing.

## About Page
- Presents the Notion-powered "About Me" document alongside contact options.
- Keeps profile information in sync with the original Notion source.

## Guestbook
- Visitors can leave public or private messages for Meti.
- Submissions flow into the Notion guestbook database so the owner can manage them centrally.
- The guestbook page shows existing entries and encourages new messages.

## Contact & Notifications
- Contact section provides quick links for reaching Meti.
- A notification form sends email alerts so the owner never misses important feedback.

## Theming & UI Polish
- Light and dark themes adjust automatically or per visitor choice.
- Shared UI elements (tooltips, headers) ensure a consistent look and feel across the site.

## Analytics & SEO
- Vercel Analytics and Speed Insights give visibility into traffic and performance.
- SEO metadata, canonical URLs, and sitemap support organic discoverability.

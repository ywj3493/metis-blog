# Site 도메인 — 유저 스토리

## US-01: 테마 전환

**독자로서**,
**라이트와 다크 모드 간 전환하고 싶습니다**,
**주변 조명에 관계없이 블로그를 편하게 읽을 수 있도록**.

### 완료 기준

- [ ] **AC-US01-01**: 페이지가 로드된 상태에서, 테마 토글을 클릭하면, 테마가 즉시 변경됨
- [ ] **AC-US01-02**: 페이지를 새로고침하면, 페이지가 로드될 때, 마지막 테마 선택이 유지됨
- [ ] **AC-US01-03**: 테마를 선택하지 않은 상태에서, 페이지가 로드되면, 시스템 테마가 기본값으로 사용됨
- [ ] **AC-US01-04**: 컴포넌트가 마운트 중일 때, 하이드레이션이 실행되면, 미스매치 방지를 위해 로딩 점 표시

## US-02: 사이트 네비게이션

**독자로서**,
**상단 네비게이션 바에서 주요 섹션에 접근하고 싶습니다**,
**About, Posts, Guestbook 사이를 빠르게 이동할 수 있도록**.

### 완료 기준

- [ ] **AC-US02-01**: 어떤 페이지에서든, 상단을 보면, 로고, 네비게이션 메뉴, 테마 토글이 보임
- [ ] **AC-US02-02**: "소개", "방명록", "포스트"를 클릭하면, 링크가 따라갈 때, 올바른 페이지로 이동함

## US-03: 프로필 및 About 조회

**독자로서**,
**홈 hero와 About 페이지를 통해 블로그 소유자를 알고 싶습니다**,
**누가 콘텐츠를 작성하는지 이해할 수 있도록**.

### 완료 기준

- [ ] **AC-US03-01**: `/`를 방문하면, 페이지가 로드될 때, 마스코트, 인사말, GitHub 링크가 있는 Hero가 보임
- [ ] **AC-US03-02**: `/about`을 방문하면, 페이지가 로드될 때, Notion 렌더링된 프로필 콘텐츠가 보임
- [ ] **AC-US03-03**: `/about`에 있을 때, 스크롤하면, 이메일과 소셜 링크가 있는 Contact 섹션이 보임

## US-04: SEO 및 탐색

**검색 엔진으로서**,
**사이트맵을 통해 모든 블로그 콘텐츠를 발견하고 robots 규칙을 따르고 싶습니다**,
**블로그가 올바르게 인덱싱되도록**.

### 완료 기준

- [ ] **AC-US04-01**: 크롤러가 `/sitemap.xml`을 요청하면, 라우트가 해석될 때, 모든 포스트 URL이 포함된 XML 반환
- [ ] **AC-US04-02**: 크롤러가 `/robots.txt`를 요청하면, 라우트가 해석될 때, 사이트맵을 가리키는 규칙 반환

# Guestbook 도메인 — 유저 스토리

## US-01: 방명록 조회

**사용자로서**,
**방명록 페이지에서 방명록 목록을 보고 싶습니다**,
**다른 방문자들이 남긴 메시지를 읽을 수 있도록**.

### 완료 기준

- [ ] **AC-US01-01**: 방명록 항목이 있을 때, `/guestbooks`를 방문하면, 날짜 내림차순으로 정렬된 항목이 표시됨
- [ ] **AC-US01-02**: 비공개 항목이 있을 때, 목록이 렌더링되면, 실제 내용 대신 "비공개 방명록 입니다." 표시
- [ ] **AC-US01-03**: 항목이 로드되면, 공개 항목을 볼 때, 작성자, 날짜, 내용이 표시됨

## US-02: 방명록 작성

**사용자로서**,
**방명록 폼을 통해 메시지를 제출하고 싶습니다**,
**블로그 소유자에게 메모를 남길 수 있도록**.

### 완료 기준

- [ ] **AC-US02-01**: 이름과 내용을 입력하고, 폼을 제출하면, Notion에 항목이 생성되고 목록에 표시됨
- [ ] **AC-US02-02**: 비공개 옵션을 체크하고, 제출하면, 상태 "비공개"로 항목 생성
- [ ] **AC-US02-03**: 이름이나 내용이 비어있을 때, 제출하면, API가 400 에러 반환
- [ ] **AC-US02-04**: 제출이 진행 중일 때, 대기하면, 로딩 스피너 표시

## US-03: 이메일 알림 수신

**블로그 소유자로서**,
**누군가 방명록을 남기면 이메일을 받고 싶습니다**,
**적시에 응답할 수 있도록**.

### 완료 기준

- [ ] **AC-US03-01**: 방명록 항목이 생성되면, 제출 성공 시, 설정된 주소로 이메일 전송
- [ ] **AC-US03-02**: 이메일 전송이 실패하면, 알람 API 에러 시, 방명록 항목은 저장된 상태 유지 (롤백 없음)

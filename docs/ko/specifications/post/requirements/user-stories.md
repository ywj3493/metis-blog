# Post 도메인 — 유저 스토리

## US-01: 포스트 탐색

**사용자로서**,
**홈 페이지와 포스트 페이지에서 블로그 포스트 그리드를 보고 싶습니다**,
**관심 있는 콘텐츠를 발견할 수 있도록**.

### 완료 기준

- [ ] **AC-US01-01**: Notion에 공개 포스트가 있을 때, 홈 페이지를 방문하면, 날짜순으로 정렬된 반응형 그리드에 모든 포스트가 표시됨
- [ ] **AC-US01-02**: 포스트에 커버 이미지가 있을 때, 포스트 카드가 렌더링되면, 적절한 비율로 커버 이미지가 표시됨
- [ ] **AC-US01-03**: 포스트에 AI 요약이 있을 때, 포스트 카드가 렌더링되면, 카드에 요약이 표시됨

| NFR | 지표 |
|-----|------|
| 페이지 로드 | 3G에서 3초 미만 |
| ISR | 180초 재검증 |

## US-02: 포스트 읽기

**사용자로서**,
**포스트 카드를 클릭하면 전체 Notion 콘텐츠를 보고 싶습니다**,
**적절한 포맷으로 전체 글을 읽을 수 있도록**.

### 완료 기준

- [ ] **AC-US02-01**: 포스트 카드를 클릭하면, 상세 페이지가 로드되고, 코드 블록, 이미지, 수식이 포함된 전체 Notion 콘텐츠가 렌더링됨
- [ ] **AC-US02-02**: 다크 모드가 활성화된 상태에서, 포스트를 볼 때, 다크 테마로 콘텐츠가 렌더링됨
- [ ] **AC-US02-03**: 포스트 URL이 slug를 사용할 때, `/posts/<slug>`에 접근하면, 올바른 포스트가 해석되어 표시됨

| NFR | 지표 |
|-----|------|
| SSG | 빌드 시 모든 포스트 사전 렌더링 |
| SEO | 메타데이터에 제목과 설명 포함 |

## US-03: 태그별 포스트 필터링

**사용자로서**,
**하나 이상의 태그를 선택해 포스트를 필터링하고 싶습니다**,
**특정 주제의 포스트를 찾을 수 있도록**.

### 완료 기준

- [ ] **AC-US03-01**: 포스트 페이지에서, 사이드바의 태그를 클릭하면, 해당 태그가 있는 포스트만 표시됨
- [ ] **AC-US03-02**: 여러 태그를 선택하면, 필터링 적용 시, 선택된 태그 중 하나라도 매칭되는 포스트가 표시됨 (OR 로직)
- [ ] **AC-US03-03**: 선택된 태그에 매칭되는 포스트가 없으면, 필터링 적용 시, 빈 상태 메시지가 표시됨
- [ ] **AC-US03-04**: 모든 태그를 해제하면, 필터링 적용 시, 모든 포스트가 다시 표시됨

| NFR | 지표 |
|-----|------|
| UX | 즉시 필터링 (네트워크 요청 없음) |

## US-04: 포스트 간 네비게이션

**사용자로서**,
**포스트 상세 페이지 하단에서 관련 포스트를 보고 싶습니다**,
**유사한 콘텐츠를 계속 읽을 수 있도록**.

### 완료 기준

- [ ] **AC-US04-01**: 포스트를 읽다가, 하단으로 스크롤하면, 유사한 태그를 공유하는 최대 4개 관련 포스트가 표시됨
- [ ] **AC-US04-02**: 관련 포스트가 표시된 상태에서, 하나를 클릭하면, 해당 포스트의 상세 페이지로 이동됨

| NFR | 지표 |
|-----|------|
| 관련성 | 공유 태그 기반, 날짜 근접성 정렬 |
